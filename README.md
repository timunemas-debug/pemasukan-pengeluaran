# Proyek Pemasukan & Pengeluaran
Script Python buat nyatet uang masuk dan keluar. Input lewat terminal, hasilnya disimpan ke Excel.
Jalankan `main.py` di terminal. Ikuti instruksi buat masukin data pemasukan atau tabungan.

- Input pemasukan
- Input tabungan
- Simpan ke saldo.xlsx
- Validasi angka dan tanggal

Proyek ini dibuat buat latihan Python dan data handling. Belum pakai GUI, tapi rencana ke depan mau ditambah.
